# Contributing to MonitoringApp

Thank you for considering contributing! Please follow these guidelines:

## How to Contribute
- Fork the repository and create your branch from `main`.
- Add tests for new features in the `tests/` folder.
- Ensure code style consistency (use ESLint/Prettier if available).
- Document new features in the README.md.
- Submit a pull request with a clear description of your changes.

## Code of Conduct
- Be respectful and constructive in all communications.
- Report issues or vulnerabilities responsibly.

## Getting Help
- Open an issue for bugs, feature requests, or questions.
